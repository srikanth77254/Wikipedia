Wikipedia Page

A simple and responsive web application that allows users to search for information on Wikipedia by entering a country name or any keyword. When the user presses Enter, the application redirects them to the corresponding Wikipedia search results page.

📌 Features
Clean and minimal user interface
Responsive design
Wikipedia search integration
Instant redirection on pressing Enter
Lightweight and easy to customize
🛠️ Technologies Used
HTML5
CSS3
JavaScript (Vanilla JS)
📂 Project Structure
Country Search Page/
│
├── Country Search page.html    # Main HTML file
├── Country Search page.css     # Styling file
├── Country Search page.js      # Search functionality
└── README.md                   # Project documentation
🚀 How It Works
Open the application in a web browser.
Enter a country name or any search keyword in the search bar.
Press Enter.
The application redirects to the Wikipedia search results page for the entered query.
💻 Installation & Usage
Clone the Repository
git clone https://github.com/your-username/country-search-page.git
Navigate to the Project Folder
cd country-search-page
Run the Application

Open the HTML file in your browser:

Country Search page.html

or simply double-click the HTML file.

Input:

India

https://en.wikipedia.org/wiki/Special:Search?search=India
